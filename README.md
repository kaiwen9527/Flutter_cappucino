## 操作
1. 一開始會進入左邊的主頁，點選Welcome後會進入右邊Cappucino的購買畫面。

2. 操作者可先選擇想要的Size，再來點選Buy Now完成購買。

3. 完成購買後會出現已完成購買的提示訊息。

4. 而點選右上角的愛心符號能再回到主頁畫面。
